# omenthings
iot platform

## 特性定义
   控制平面和数据平面分开
   支持云原生指标化
   轻量化不依赖过多中间件
   支持通信/业务协议扩展（TCP,MQTT,UDP,CoAP,HTTP,GRPC,RPC,RTMP,RTSP,PLC,Modbus ｜支持 摄像头、电表、光伏/储能逆变器、充电桩等多种设备 ）
   多租户支持/支持只读和读写账户权限设置
   支持规则链数据流处理
   支持字典数据配置
   docker-compose一键部署
   支持时序数据存储（CK）
   支持事件告警通知（watchAlert）
   支持大屏数据展示
   支持设备打标实现灰度功能
